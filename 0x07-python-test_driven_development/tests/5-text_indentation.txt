The ``Text identation`` module
===========================

Using ``text_indentation``
----------------------

First import ``text_indentation`` from the ``Text identation`` module:

    >>> text_indentation = __import__("5-text_indentation").text_indentation

Now use it:

    >>> text_indentation('abc.def')
    abc.
    <BLANKLINE>
    def

    >>> text_indentation('abc.       def')
    abc.
    <BLANKLINE>
    def

    >>> text_indentation('abc      .        def.')
    abc      .
    <BLANKLINE>
    def.
    
    >>> text_indentation('abc..def')
    abc.
    <BLANKLINE>
    .
    <BLANKLINE>
    def

    >>> text_indentation('abc:def')
    abc:
    <BLANKLINE>
    def

    >>> text_indentation('abc:       def')
    abc:
    <BLANKLINE>
    def

    >>> text_indentation('abc      :        def.')
    abc      :
    <BLANKLINE>
    def.

    >>> text_indentation('abc:def.')
    abc:
    <BLANKLINE>
    def.

    >>> text_indentation('abc.def?')
    abc.
    <BLANKLINE>
    def?

    >>> text_indentation('abc?       def')
    abc?
    <BLANKLINE>
    def

    >>> text_indentation('abc      ?        def.')
    abc      ?
    <BLANKLINE>
    def.

    >>> text_indentation('abc?.def')
    abc?
    <BLANKLINE>
    .
    <BLANKLINE>
    def

    >>> text_indentation(5)
